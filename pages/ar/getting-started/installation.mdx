# **دليل التثبيت لإطار العمل كوكب Kawkab**

إطار العمل **كوكب Kawkab** يقدم لك بيئة تطوير مرنة وقوية لبناء تطبيقات ويب حديثة. في هذا الدليل، سنتناول خطوات تثبيت **كوكب Kawkab** عبر **Node.js** و **Bun**.

## **التثبيت عبر Node.js**

### **المتطلبات المبدئية:**
قبل البدء، تأكد من أنك قد قمت بتثبيت **Node.js** و **npm** على جهازك. للحصول على النسخة المطلوبة، يمكنك تحميل **Node.js** من [الموقع الرسمي](https://nodejs.org/) مع التأكد من أنك تستخدم النسخة 18.0 أو أحدث.

### **إنشاء مشروع جديد باستخدام كوكب Kawkab:**
لبدء مشروع جديد باستخدام **كوكب Kawkab**، قم بتشغيل الأمر التالي في الطرفية:

```bash
npx create-kawkab-app@latest init my-kawkab-app
```

سيقوم هذا الأمر بإنشاء مشروع جديد باسم `app` داخل مجلد منفصل يحتوي على جميع ملفات الإعدادات الخاصة بـ **كوكب Kawkab**.

### **الانتقال إلى مجلد المشروع:**
بعد أن يتم إنشاء المشروع، قم بالانتقال إلى المجلد الذي تم إنشاؤه باستخدام الأمر التالي:

```bash
cd app
```

### **تشغيل خادم التطوير:**
لتشغيل خادم التطوير المحلي الخاص بـ **كوكب Kawkab**، استخدم الأمر التالي:

```bash
npm run dev
```

سيتم تشغيل خادم التطوير على الرابط المحلي `http://localhost:3000`، مما يمكنك من معاينة تطبيقك وتجربة جميع الميزات بشكل فوري.

---

الآن أصبح لديك تطبيق جاهز! **كوكب Kawkab** يوفر لك العديد من الأدوات والميزات المدمجة التي تساعدك على بناء تطبيقات ويب عالية الجودة بسرعة وكفاءة.

## **التثبيت عبر Bun**

إذا كنت تفضل استخدام **Bun** كـ **JavaScript runtime** سريع وفعّال، يمكنك تثبيت **كوكب Kawkab** باستخدام **Bun CLI** باتباع الخطوات التالية:

### **إنشاء مشروع جديد باستخدام كوكب Kawkab عبر Bun:**

```bash
bunx create-kawkab-app@latest init my-kawkab-app
```

بقية الخطوات مشابهة تمامًا لما تم شرحه في التثبيت عبر **Node.js**.
