# ملفات البيانات باستخدام Kawkab

تُستخدم ملفات البيانات في **Kawkab** لتنظيم وتخزين البيانات الثابتة أو البيانات المعتمدة التي يتم استخدامها في تطبيقك. يمكن أن تكون هذه البيانات عبارة عن قوائم، إعدادات، أو أي معلومات ثابتة أخرى تود استخدامها في مشروعك بطريقة منظمة.

## إنشاء ملفات البيانات باستخدام Kawkab

في هذا الدليل، سنوضح كيفية إنشاء ملفات بيانات جديدة باستخدام إطار العمل **Kawkab** ودمجها في تطبيقك.

## الخطوة 1: إنشاء ملف بيانات جديد

لإنشاء ملف بيانات جديد باستخدام **Kawkab**، قم بتشغيل الأمر التالي في سطر الأوامر:

```bash
npm run kawkab data:make <name> [module]
```

### **تفاصيل المعلمات:**

- **`<name>`**: اسم ملف البيانات الذي ترغب في إنشائه (على سبيل المثال `cats`).
- **`[module]`**: اسم الموديول الذي يحتوي على البيانات (اختياري، الافتراضي هو `main`).

### مثال على كيفية تنفيذ الأمر:

```bash
npm run kawkab data:make cats
```

هذا الأمر سينشئ ملف بيانات باسم `cats` داخل الموديول `main`.

### ماذا يحدث بعد تشغيل الأمر؟

- سيتم إنشاء ملف بيانات باسم `'cats'` في الموديول المحدد.
- سيحتوي هذا الملف على البيانات التي يمكن استخدامها في التطبيق. على سبيل المثال، في هذا الدليل سيتم إنشاء ملف يحتوي على قائمة بأنواع القطط المشهورة.

## الخطوة 2: كيفية استخدام البيانات في التطبيق

بمجرد إنشاء ملف البيانات، يمكنك استيراده واستخدامه داخل ملفات المشروع الخاصة بك.

### استيراد البيانات:

لاستيراد البيانات التي تم إنشاؤها، استخدم الكود التالي في ملفك:

```javascript
import catsData from '../data/cats';
```

### محتوى ملف البيانات:

سيحتوي ملف البيانات `cats` على البيانات التالية (أنواع القطط):

```javascript
export default {
  data: [
    "Persian",        // قطط فارسية
    "Maine Coon",     // قطط مين كون
    "Bengal",         // قطط بنغالية
    "Siamese",        // قطط سيامية
    "Ragdoll",        // قطط راغدول
    "British Shorthair", // قطط شورتير البريطاني
    "Sphynx",         // قطط سفينكس
    "Abyssinian",     // قطط أبيسينيان
    "Scottish Fold",  // قطط سكوتش فولد
    "Birman"          // قطط بيرمان
  ]
}
```

### استخدام البيانات في التطبيق:

الآن، يمكنك الوصول إلى البيانات واستخدامها بسهولة في أي جزء من التطبيق:

```javascript
const data = catsData.data;
console.log(data);  // سيطبع لك قائمة بأنواع القطط
```

## ملاحظات إضافية

- تأكد من أن المسار الذي تستخدمه لاستيراد البيانات صحيح بناءً على هيكل المجلدات في مشروعك.
- يمكنك استخدام نفس الأسلوب لإنشاء ملفات بيانات أخرى بتغيير اسم الملف الذي تم إنشاؤه (مثل `dogs` بدلاً من `cats`).
  
- إذا كنت تستخدم أكثر من موديول، يمكنك تحديد الموديول الذي يحتوي على البيانات باستخدام المعامل `[module]`:

```bash
npm run kawkab data:make cats customModuleName
```

## الخاتمة

إنشاء ملفات البيانات باستخدام Kawkab يساهم في تنظيم وتبسيط إدارة البيانات الثابتة في تطبيقك. بفضل Kawkab، يمكنك إنشاء هذه الملفات بسهولة ودمجها في تطبيقك بسرعة. استمتع باستخدام **Kawkab** لتطوير تطبيقاتك بشكل أكثر كفاءة ومرونة!
